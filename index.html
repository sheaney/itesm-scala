Scala es un lenguaje de programación que busca mezclar las mejores prácticas y técnicas de programación funcional y programación orientada a objetos. Te permite escribir código sencillo y reusable e interoperar fácilmente con todas las librerías de Java.

Scala es un lenguaje que ha estado ganando popularidad lentamente en la industria, con compañías como Twitter, LinkedIn, Coursera, entre otras usándolo para algunos de sus servicios.

Este curso tiene el objetivo de enseñarte los conceptos básicos de Scala para que puedas codificar programas sencillos y tengas las bases para explorar el lenguaje más a fondo.

Preguntas Frecuentes

¿Cuánto cuesta?
El curso tiene un costo de $50 pesos. Lo que se recaude irá al fondo de becas de la SAITC.
¿Dónde puedo pagar?
Se paga el mismo día en el salón donde se darán las sesiones.
¿Para quién es este curso?
Este curso es para gente que tiene 1 a 2 años programando ya sea en Java, C, Ruby, Python, Javascript o cualquier otro lenguaje. Sin embargo, para aprovechar bien este curso, ayuda tener experiencia con programación orientada a objetos y/o programación funcional.
¿Qué necesito llevar?
Para este curso solo necesitas llevar tu laptop y tener listo tu editor preferido de texto. La instalación se hará en el salón. Lo más importante es que vengas con toda la disposición para aprender y preguntar mucho!.
